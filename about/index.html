About this site

Xander's Lander is an independent publication launched in April 2023 by Alex Rudolph.  Due to the limitations of running ghost in a github page, authentication doesn't work because of a lack of a live database. For security reasons, using an open source GitHub page would expose connection string that could be used to exploit vulnerabilities.

Fresh content, un-delivered

Everything in this site is designed to be free and open. For that reason, I will not add any features or functions that would require any financial investment. Anyone who has the technical aptitude can perform this same setup with very low effort.
